# Coqui TTS (MPL-2.0 License - safe for commercial use)
# Enables local TTS processing without cloud dependency.
# Note: Requires significant disk space (~2GB) and GPU recommended for speed.
#
# To install Coqui TTS with subtitle support:
#   pip install coqui-tts faster-whisper
#
# Or install with CUDA support (NVIDIA GPUs):
#   pip install coqui-tts[cuda] faster-whisper
#
# GPU Acceleration Support:
#   - NVIDIA: CUDA (automatic detection)
#   - Apple Silicon (M1/M2/M3): MPS (automatic detection)
#   - No GPU: Falls back to CPU
#
# IMPORTANT: faster-whisper is REQUIRED for accurate word-level subtitles!
# Without it, subtitles will use estimated timing (less accurate).
# faster-whisper runs 100% locally (MIT license, no data sent to servers)
# Models download on first use: tiny (~75MB), base (~150MB), small (~500MB)
#
# ============================================================================

#   - coqui-tts (pip install coqui-tts)
#   - faster-whisper (pip install faster-whisper) - REQUIRED for subtitles
#   - A voice sample audio file (WAV, MP3, FLAC, OGG, or M4A)
#   - 5-30 seconds of clear speech with single speaker
#   - Minimal background noise
#
# To install all voice cloning dependencies (recommended):
#   pip install coqui-tts faster-whisper
#
